eCommerce PHP-MySQL

Este sitio ha sido el primero que he elaborado a modo de prueba. Se trata de un sitio únicamente para personas que intenten editar una base de datos MySQL desde el sitio. Se pueden agregar/quitar/modificar Marcas, Categorías, Productos, stock, precio; con la necesidad de loguearse para tener acceso. Está elaborado en lenguaje PHP, HTML, CSS, un poco de Javascript y utilizando Bootstrap. Como se puede ver en las imágenes, es el back-end de un sitio web que contenga un carrito de compras ligado a una base de datos MySQL en constante conexión.
